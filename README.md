# CodeAlpha_Calculator

A premium, modern, and highly responsive web-based Calculator designed with a clean **Neumorphic (Soft UI)** aesthetic. This project was developed as part of my Front-End Development Internship at **CodeAlpha**.

---

## 🚀 Features

*   **Neumorphic Design:** Premium gray-toned color palette with soft, realistic shadows for an elegant tactile feel.
*   **Live Preview:** Displays a real-time smart preview of the calculation result directly below the main input field.
*   **Intuitive Layout:** Clean, grid-based interface designed for a smooth and natural user experience. 
*   **Fully Responsive:** Completely optimized for seamless use across mobile screens, tablets, and desktops.

---

## 🛠️ Tech Stack Used

*   **HTML:** Structured semantic layout.
*   **CSS:** Custom styling, variables, grid layouts, and smooth transition animations.
*   **JavaScript:** Dynamic logic for arithmetic evaluations, real-time live preview tracking, and error handling.
